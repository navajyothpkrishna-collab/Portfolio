Personal Portfolio Website

About the Project

This is my personal portfolio website built to showcase my skills, projects, and interests as an aspiring software engineer. The website features a modern dark theme, smooth animations, and an interactive 3D background.

Features

Modern dark UI with gradient design Smooth scrolling and animations Skills section highlighting technical expertise Projects showcase Social media integration (Instagram, GitHub, LinkedIn) Contact section with email support 3D background using Three.js

Built With

HTML5 – Structure CSS3 – Styling & animations JavaScript – Interactivity Three.js– 3D background effects
